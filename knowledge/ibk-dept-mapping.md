## 규제·제재 주제 → IBK 담당 부서 매핑

> FSS 제재/경영유의 지적사항의 주제 키워드 → IBK 담당 부서(주담당). 상세·협조부서는 정본 참조.

| 주제 키워드 | IBK 담당 부서 (주담당) | 소속 그룹 |
|---|---|---|
| 여신·대출·LTV·DSR·보증·여신금리 | 여신기획부 | 여신운영그룹 |
| 채권관리·추심·채무조정·신용회복 | 여신관리부 | 여신운영그룹 |
| 자금세탁·KYC·CDD·FIU·특금법·AML | 자금세탁방지부 | 준법감시인 |
| 개인정보·신용정보 (관리·보호 규정) | 정보보호총괄부 | 리스크관리그룹 |
| 마이데이터·데이터 활용 | 데이터혁신부 | 디지털그룹 |
| 전자금융·오픈뱅킹·비대면 (규정 소관) | 개인디지털사업부 (IT내부통제부 보안 협조) | 개인고객그룹 |
| IT보안·정보보호규정·사이버 | 정보보호총괄부 · IT내부통제부 | 리스크관리그룹·IT그룹 |
| 내부통제·지배구조·준법감시·책무구조도 | 준법지원부 · 내부통제총괄부 | 준법감시인 |
| 금융복합기업집단·지주·경영전략·ESG | 전략기획부 | 경영전략그룹 |
| 금융소비자보호·불완전판매·민원·광고/상품공시 | 금융소비자보호부 (광고·상품공시는 준법지원부) | 금융소비자보호그룹 |
| 투자일임·펀드·신탁·WM·자산관리 | 자산관리사업부 | 자산관리그룹 |
| 카드·카드한도·카드모니터링 | 카드사업부 | 카드사업그룹 |
| 기타 | 내부통제총괄부 | 준법감시인 |

## ❌ 존재하지 않는 부서명 (LLM 출력 시 수정 필요)

| 잘못된 부서명 | 올바른 부서명 | 비고 |
|---|---|---|
| 법무실 | 준법지원부 | IBK에 법무실 없음 |
| 개인정보보호팀 | 정보보호총괄부 | 팀 단위 없음. 개인·신용정보 관리·보호 규정 소관은 정보보호총괄부 |
| AML팀 | 자금세탁방지부 | 정식 부서명 사용 |
| IT운영부 | IT내부통제부 | IT운영본부·IT시스템운영부 등 존재 |
| 경영전략부 | 전략기획부 | 경영전략그룹 내 전략기획부가 정식 명칭 |
| 소비자보호부 | 금융소비자보호부 | 그룹 및 부서명 모두 공식 명칭 사용 |

## IBK 조직 전체 구조 (부서 레벨)

```
준법감시인
  ├─ 준법지원부
  ├─ 내부통제총괄부
  ├─ AML보고책임자
  └─ 자금세탁방지부

여신운영그룹
  ├─ 여신기획부
  ├─ 여신심사부 / 여신심사센터
  ├─ 여신관리부
  ├─ 기업개선부
  └─ 사모투자부

IT그룹
  ├─ IT기획부
  ├─ IT내부통제부
  ├─ IT운영본부 / IT시스템운영부
  └─ IT개발본부 / IT금융개발부 / IT디지털개발부

리스크관리그룹
  ├─ 리스크총괄부 / 리스크관리부
  ├─ 정보보호최고책임자(CISO)
  ├─ 정보보호총괄부
  └─ 정보보호운영부

금융소비자보호그룹
  ├─ 금융소비자보호부
  └─ 금융소비자자원부

자산관리그룹
  ├─ 자산관리사업부
  ├─ WM사업부
  ├─ 신탁부
  └─ 수탁사업부

경영전략그룹
  ├─ 전략기획부
  ├─ ESG경영부
  ├─ 경영관리부
  └─ 브랜드전략부 등

카드사업그룹
  ├─ 카드사업부
  ├─ 카드디지털사업부
  └─ 카드지원부
```
