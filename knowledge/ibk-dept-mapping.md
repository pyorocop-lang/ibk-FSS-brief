# IBK기업은행 부서 매핑 기준

> 조직 정본: 2026-07-14 「직제규정」 전문 및 직제도
> 업무 매핑 원천: 2026-06-23 내규관리 공식 목록 PDF 기반 기존 매핑
> 상태: 조직명·소속관계 갱신 완료 / 최신 「업무분장규정」 미반영
> 용도: `analyst.js` 시스템 프롬프트 및 fallback 부서 배정 기준

## 사용 제한

- 이 파일의 조직명과 소속관계는 2026-07-14 기준자료로 확인했다.
- 세부 업무소관은 「직제규정」 제3조제2항에 따라 「업무분장규정」이 정한다.
- 아래 잠정 매핑은 과거 공식 내규 소관정보에서 부서명이 현재도 존속하는 경우에만 유지한다.
- 현행 직제에서 사라진 부서의 업무를 새 부서로 자동 승계시키지 않는다.
- 업무분장 미확인 주제는 `내부통제총괄부` 등 임의의 기본부서로 보내지 말고 `소관부서 확인 필요`로 처리한다.

## 자동 배정 보류 주제

아래 주제는 과거 소관부서가 현행 직제에서 확인되지 않거나 조직개편의 영향이 커서 최신 업무분장 확인 전 자동 배정하지 않는다.

| 주제 | 과거 매핑 또는 문제 | 처리 |
|---|---|---|
| 마이데이터·데이터 활용 | 데이터혁신부·데이터본부·디지털그룹이 현행 직제에서 확인되지 않음 | 소관부서 확인 필요 |
| AI 윤리·AI 거버넌스 | AI&Tech센터·디지털혁신부가 현행 직제에서 확인되지 않음 | 소관부서 확인 필요 |
| 경영전략·지식재산권 | 과거 전략기획부가 현행 직제에서 확인되지 않음 | 경영전략부로 임의 승계 금지 |
| 회계·내부회계·공시 | 과거 재무회계부가 현행 직제에서 확인되지 않음 | 소관부서 확인 필요 |
| BCP·비상계획·물리보안 | 과거 안전기획팀·비상계획팀이 현행 직제에서 확인되지 않음 | 안전기획부로 임의 승계 금지 |
| 방카슈랑스 | 과거 방카슈랑스사업부가 현행 직제에서 확인되지 않음 | 소관부서 확인 필요 |
