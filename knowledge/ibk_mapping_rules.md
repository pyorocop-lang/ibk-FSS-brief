# IBK 내규 → 업무영역 → 담당 부서 매핑

> 조직 기준: 2026-07-14 「직제규정」 전문 및 직제도
> 내규 소관 기준: 2026-06-23 내규관리 공식 목록 PDF 기반 기존 자료
> 갱신일: 2026-07-15
> 상태: 현행 조직명 검증 완료 / 최신 「업무분장규정」 미반영

## 1. 적용 원칙

1. 조직의 존재와 소속은 2026-07-14 직제자료를 따른다.
2. 업무소관은 「직제규정」 제3조제2항에 따라 별도 「업무분장규정」이 정한다.
3. 과거 내규 소관부서가 현행 직제에도 같은 명칭으로 존재하는 경우에만 잠정 매핑을 유지한다.
4. 과거 소관부서가 폐지·변경된 경우 새 부서로 자동 승계하지 않는다.
5. `소관 확인 필요` 항목은 LLM의 추론만으로 `dept`를 확정하지 않는다.
6. 관련 내규가 여러 부서에 걸치면 내규별 소관을 먼저 확인하고 주담당·협조부서를 구분한다.

## 5. LLM 적용 순서

1. 지적 주제를 제3절의 외부 이슈 영역과 대조한다.
2. `유지` 항목은 현행 조직 정본에서 부서 존재를 다시 확인한다.
3. `내규별 구분 필요` 항목은 지적사항과 직접 관련된 내규 소관부서를 주담당으로 선택한다.
4. 제4절 항목은 부서명을 추론하지 않고 `소관부서 확인 필요`로 남긴다.
5. 협조부서는 표에 명시된 부서만 후보로 사용하며 근거가 없으면 빈 배열로 둔다.
